# 📊 Organizational Project Management System

A Full-Stack Project Management Application built using React, Node.js, Express, and MongoDB.

---

# 📖 Overview

The **Organizational Project Management System** is a production-ready full-stack web application designed to manage the complete lifecycle of organizational projects.

The system allows administrators and project managers to create projects, estimate costs, assign team members, manage tasks, track working hours, monitor employee performance, generate reports, and complete project delivery—all through a secure and responsive interface.
